# 🎓 College Analysis Dashboard | Power BI

## 📌 Project Overview

The **College Analysis Dashboard** is an interactive Power BI project developed to analyze academic performance, admissions, student demographics, faculty performance, campus resources, and institutional outcomes. It provides a comprehensive view of college operations through interactive dashboards, helping administrators and stakeholders make informed, data-driven decisions.

The dashboard consists of **20 interactive report pages**, covering various academic and administrative aspects of the institution.

## 💡 Business Insights

This dashboard enables educational institutions to:

- Monitor academic performance across all courses.
- Analyze student demographics and enrollment trends.
- Track admission statistics over multiple academic years.
- Evaluate faculty and staff performance.
- Measure staff feedback and student satisfaction.
- Monitor campus resources and institutional events.
- Analyze student retention and graduation rates.
- Support strategic decision-making using interactive visualizations.
