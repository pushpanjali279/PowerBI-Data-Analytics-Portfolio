# 💼 Job Market Analysis Dashboard | Power BI

## 📌 Project Overview

The **Job Market Analysis Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI** to analyze job market trends and provide valuable insights into hiring patterns, salary ranges, job benefits, company sizes, job types, and job locations.

The dashboard transforms raw job market data into meaningful visualizations, enabling users to explore employment trends and make data-driven career and business decisions.

---

## 📖 Extended Description

The primary objective of this project was to analyze job market data and uncover insights into recruitment trends, compensation, benefits, company characteristics, and employment opportunities across different locations.

The dataset was cleaned and transformed using **Power Query** by removing duplicate records, handling missing values, correcting data inconsistencies, and preparing the data for analysis. After data preparation, the dataset was imported into **Power BI**, where data modeling techniques and **DAX measures** were used to create interactive reports.

The dashboard includes multiple report pages featuring dynamic filters, slicers, KPI cards, charts, and visualizations that allow users to explore job market trends efficiently.

This project demonstrates practical skills in **Power BI, Power Query, DAX, Data Modeling, Dashboard Design, and Data Visualization**.
